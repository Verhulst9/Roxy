## 模块概述

### 职责定位
Frontend 模块是 Nakari 的**用户交互界面**，负责聊天、语音、视觉等交互方式的展示和用户输入处理。该模块是用户与 Nakari 交互的主要入口。

### 设计目标
1. **流畅交互**：保证交互的流畅性和响应速度
2. **美观界面**：提供美观、易用的用户界面
3. **多端支持**：支持 Web、桌面、移动端
4. **个性化体验**：支持主题切换、个性化设置
5. **低延迟**：实时响应，延迟控制在 500ms 以内

### 技术选型

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| React | 18+ | 成熟的前端框架，生态丰富 |
| Next.js | 14+ | 全栈框架，支持 SSR 和 API 路由 |
| Tailwind CSS | 3.4+ | 快速构建 UI，高度可定制 |
| WebSocket | API | 实时通信，低延迟 |
| WebRTC | API | 实时音视频通信 |

## 核心组件

### 1. Chat UI（聊天界面）

**职责**：
- 展示聊天对话
- 处理用户输入
- 消息历史管理

**接口设计**：
```typescript
interface ChatUIProps {
  onSendMessage: (message: string) => void;
  onVoiceInput: () => void;
  onClearHistory: () => void;
}

interface Message {
  id: string;
  role: 'user' | 'assistant' | 'system';
  content: string;
  timestamp: Date;
  metadata?: Record<string, any>;
}
```

### 2. Audio UI（语音交互界面）

**职责**：
- 语音输入按钮
- 音量指示器
- 音色选择

**接口设计**：
```typescript
interface AudioUIProps {
  onStartListening: () => void;
  onStopListening: () => void;
  isListening: boolean;
  volume: number;
  availableVoices: VoiceInfo[];
  selectedVoice: string;
  onSelectVoice: (voiceId: string) => void;
}

interface VoiceInfo {
  id: string;
  name: string;
  gender: 'male' | 'female' | 'neutral';
  language: string;
}
```

### 3. Vision UI（视觉展示界面）

**职责**：
- 摄像头画面展示
- 视觉分析结果展示
- 图像上传

**接口设计**：
```typescript
interface VisionUIProps {
  onStartCamera: () => void;
  onStopCamera: () => void;
  isCameraActive: boolean;
  onUploadImage: (file: File) => void;
  analysisResult?: AnalysisResult;
}

interface AnalysisResult {
  faces: Face[];
  emotions: Emotion[];
  objects: string[];
  description: string;
}
```

```typescript
interface FrontendModule {
  /**
   * 发送消息

  sendMessage(message: string): Promise<Response>;

  /**
   * 开始语音输入

  startListening(): void;

  /**
   * 停止语音输入
   */
  stopListening(): void;

  /**
   * 开始摄像头
   */
  startCamera(): Promise<void>;

  /**
   * 停止摄像头
   */
  stopCamera(): void;

  /**
   * 上传图像
   */
  uploadImage(file: File): Promise<void>;

  /**
   * 连接到 Nakari 后端
   */
  connect(url: string): Promise<void>;

  /**
   * 断开连接
   */
  disconnect(): void;
}
```

该模块为 Nakari 提供了完整的用户交互界面，让用户能够通过聊天、语音、视觉等多种方式与 Nakari 进行自然、流畅的交互。
