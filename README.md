## 后端状态

### 已实现的 Live2D 工具 (4 个)
| 工具名称 | 功能 |
|---------|------|
| `live2d_set_motion` | 设置 Live2D 动画 |
| `live2d_set_emotion` | 设置 Live2D 表情 |
| `live2d_analyze_emotion` | 分析文本情感 |
| `live2d_set_param` | 直接设置参数 |

### 已实现的状态类型
- `idle` - 闲置状态
- `thinking` - 思考状态
- `speaking` - 说话状态
- `processing` - 处理状态

### 已实现的情感类型
- `neutral` - 中性
- `happy` - 开心
- `sad` - 难过
- `angry` - 生气
- `surprised` - 惊讶
